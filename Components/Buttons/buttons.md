
# Buttons

A collection of button UI components based on proportional spatial relationships that determine sizes, dimensions, spacing, and typography while maintaining visual consistency.

## Classes

### Sizes

.ct-btn        → Default size

.ct-btn-xl     → Extra-large button

.ct-btn-lg     → Large button

.ct-btn-md     → Medium button

.ct-btn-sm     → Small button

.ct-btn-xs     → Extra-small button

### Orientation

.btn-vertical  → Stacks button content vertically

### Content

.btn-label     → Main text label

.btn-graphic   → Graphic element (e.g., image or SVG)

.btn-glyph     → Font icon or Unicode character

### Radius

.btn-radius-pill → Fully rounded pill-style button
