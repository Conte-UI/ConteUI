# Misc Utilities

This section includes additional utilities that do not belong to a specific category.

---

## Visual Effects

shadow   → box-shadow  
opacity  → opacity  
filter   → filter  

**Example:**
```html
<div class="shadow:0_4px_10px_rgba(0,0,0,0.2) opacity:0.8"></div>
````

---

## Transform & Motion

transform  → transform
scale      → scale
transition → transition
animation  → animation

**Example:**

```html id="u3z5k1"
<div class="transform:rotate(10deg) transition:all_0.3s"></div>
```

---

## Layout Helpers

overflow → overflow
float    → float
z-index  → z-index
ratio    → aspect-ratio
box      → box-sizing
v-align  → vertical-align

**Example:**

```html id="k9ql7d"
<div class="overflow:hidden z-index:10"></div>
```

---

## Behavior

scroll → scroll-behavior
dir    → direction
caret  → caret-color

**Example:**

```html id="d2x8fj"
<div class="scroll:smooth"></div>
```

---

## Graphics (SVG)

fill   → fill
stroke → stroke

**Example:**

```html id="v8r1tp"
<svg class="fill:red stroke:black"></svg>
```

---

## Notes

* These utilities provide additional control over rendering and behavior.
* All values are dynamic and accept raw CSS syntax.
