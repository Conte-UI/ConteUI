
# **Display**

## Layout
d-block         → display: block  
d-inline        → display: inline  
d-inline-block  → display: inline-block  
d-contents      → display: contents  

## Flex & Grid
d-flex          → display: flex  
d-inline-flex   → display: inline-flex  
d-grid          → display: grid  
d-inline-grid   → display: inline-grid  

## Table & Ruby
d-table         → display: table  
d-ruby          → display: ruby  

## Visibility
d-none          → display: none  

## Global Values
d-inherit       → display: inherit  
d-initial       → display: initial  
d-revert        → display: revert  
d-revert-layer  → display: revert-layer  
d-unset         → display: unset  

**example**
```html
<div class="d-flex p:20px bg:#eee">
  <div class="w:100px h:100px bg:green"></div>
  <div class="w:100px h:100px bg:blue"></div>
</div>
```
